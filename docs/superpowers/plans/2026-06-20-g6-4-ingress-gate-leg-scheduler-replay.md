**Goal:** Build the per-adapter admission-control + fair-multiplexing + bounded-buffering layer so N gateway-hosted adapter legs (+ the TUI dial-in leg) safely coexist over the single gateway↔core link: a coarse **payload-blind per-adapter ingress gate** (sustained-rate token bucket + in-flight cap) that back-pressures on trip; a **fair leg scheduler** so no chatty/large-payload leg starves another or the live TUI; and **one ReplayBuffer per leg** under a **global aggregate resident-byte cap** so total pre-DLP T1 in the always-up SETUID process is bounded regardless of N. Plus per-adapter `{adapter}`-labelled metrics + the release-blocking per-leg restart-survival integration test.

**Architecture:** Three new gateway collaborators wired in the gateway process, NOT in the core: `PerAdapterIngressGate` (per leg — two-tier token-bucket + concurrency cap; on trip → back-pressure + `gateway_ingress_throttled_total{adapter}` + audit; in-flight slot released on TTL-evict so a stalled leg can't wedge); `GatewayLegScheduler` (fair round-robin/credit scheduling of the N legs' + TUI's egress over `core_link`'s single writer, bounded per-leg send queue, isolated per-leg tasks — no HoL); a `GlobalReplayCap` coordinator that bounds the SUM of all legs' `ReplayBuffer` resident bytes (each leg reuses the unchanged Spec A `ReplayBuffer`, keyed on `adapter_id`). The gateway stays payload-blind for T3 bodies (only the out-of-band seq/ack/dedup/`adapter_id` envelope is parsed; the per-leg buffer/relay treat bodies as opaque — no parse, no log). The core's `_PreResolutionLimiter` is NOT touched (kept as per-id defense-in-depth; the gateway gate is additive volumetric — the explicit comms-F2 overturn). Real hosted adapter children are G6-5; G6-4 builds + tests against the existing placeholder factory / fake-core harness.

**Tech Stack:** Python 3.12+, asyncio (per-leg tasks, fair scheduler), structlog + prometheus_client (per-adapter labelled series), pytest + pytest-asyncio (fake clock for the token-bucket/TTL + scheduler-fairness; Spec A's fake-core restart harness for the per-leg restart-survival integration test). Pure gateway-side; runs non-root.
